# Takudzwanashe Musere Portfolio

Personal portfolio website for **Takudzwanashe Musere**, a full-stack developer and IT professional based in Harare, Zimbabwe.

The site presents my background, technical skills, featured projects, resume, certifications, and contact details in a responsive single-page layout.

## Live Sections

- **Home** - introduction, profile image, CV download, and social links
- **About** - short professional summary and technical skills
- **Projects** - featured work including school systems, ReelScholar, and offline-first desktop software
- **Resume** - education, work experience, and certifications
- **Contact** - email, phone, location, social links, and contact form

## Featured Skills

- Flutter and React / React Native
- Python, FastAPI, JavaScript, HTML, and CSS
- Firebase, MySQL, MongoDB, and SQLite
- Machine learning and AI content filtering
- Cybersecurity and network security monitoring
- Git and GitHub

## Projects Highlighted

### School Portal System

A full-stack school portal with role-based access for admins, staff, and students. Built with Flutter Web, FastAPI, and Firebase.

### ReelScholar

An AI-powered microlearning mobile app for university students, featuring custom-trained content filtering, real-time messaging, quizzes, and personalized learning content.

### Divine Child Primary School Management System

An offline-first desktop school management system built with Electron.js, Node.js, and SQLite for managing students, fees, attendance, academics, and reports.

## Run Locally

Open `index.html` directly in a browser, or use a local development server:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```
